# AI-Resume-Screening-System
AI-powered resume screening system using NLP, TF-IDF, Streamlit and Python.
## About the Project
Recruiters often receive hundreds of resumes for a single job opening, making the screening process time-consuming and repetitive.
To understand how AI and Machine Learning can help in recruitment, I built an AI Resume Screening System that analyzes resumes, extracts important skills, and compares them with job requirements to calculate a match score.
The project combines web scraping, data analysis, NLP, machine learning, and dashboard development into a single end-to-end workflow.
# What This Project Does
* Collects job-related data using web scraping
* Analyzes hiring trends through Exploratory Data Analysis (EDA)
* Extracts text from PDF resumes
* Identifies candidate skills automatically
* Compares resumes with job requirements using NLP techniques
* Calculates a resume-job match score
* Highlights missing skills
* Displays results through an interactive Streamlit dashboard
